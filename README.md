# Dose Certa

> Um app de controle de medicamentos pessoais

## 🎯 Problema
Muitas pessoas esquecem de tomar seus remedios na hora certa ou a finalidade dele (Em especial, os idosos que tomam muitos remedios). Quando vão passar em consultas e falar sobre os medicamentos que toma, a maioria tem um branco constrangedor. 

## 💡 Solução
Com o Dose certa, essas pessoas terão lembretes para tomar os remedios, porque estão tomando aquele remedio, além das informações gerais (dosagems, quantidade, duração, etc)
